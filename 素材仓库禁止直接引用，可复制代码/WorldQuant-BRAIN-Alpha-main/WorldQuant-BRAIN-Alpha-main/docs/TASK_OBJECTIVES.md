**完成标准**：零硬编码凭据、CI 秘密扫描全覆盖

| 子目标 | 关键结果 | 验证方式 |
|--------|----------|----------|
| 1.1 清理硬编码凭据 | 测试脚本、截图文件中无明文密码 | 旧账号片段和旧密码片段全仓扫描返回空 |
| 1.2 完善 .gitignore | `data/e2e_screenshots/` 已排除 | `git status` 不显示该目录 |
| 1.3 扩展 CI 扫描范围 | 测试目录纳入秘密扫描 | CI 使用 `--include-all` 参数 |
| 1.4 账号轮换 | 旧凭据失效 | 手动验证登录失败 |

**执行步骤**：
```bash

# 2. 更新 .gitignore

# tests/qa_e2e_new_user_walkthrough.py
# scripts/run_e2e_walkthrough.py

# .github/workflows/quality-gate.yml 第39行添加 --include-all
```
